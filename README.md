# choza-API
Choza API START

- 1.- Follow the general flask installation 
    [flask docs](https://flask.palletsprojects.com/en/1.1.x/installation/#installation)

- 2. Export the env variables : 
    ```
    FLASK_APP=app/app.py 
    FLASK_ENV=development 
    ```
- 3.- Make a flask run : 

    flask run
